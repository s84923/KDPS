プル
git pull

ファイルのステージ(一覧用)
git add .

コミット
git commit -m "コミットメッセージ"

プッシュ
git push

手順
１.プル
２.ステージ
３.コミット
４.プッシュ
(プッシュとコミットは、衝突の可能性あるから、一声かけてやろう！)


モジュールのimportについて
多分、オーナー以外も出来ると思う。

手順
１.必要なモジュールをpipする。
２.pip freeze -> requirements.txt　このコマンドを実行する
３.終わり(requirements.txtにモジュール情報が保存される)
４.インストールする場合
５.pip install -r requirements.txt


現在の変更をコミットまたはスタッシュ：
git commit -am "Save changes"
# または
git stash

リモートブランチを手動でマージ：
git pull --no-rebase

自分の変更をリセットして最新の状態にするときは
git reset --hard
git pull
